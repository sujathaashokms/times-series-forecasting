# Time Series Forecasting – Wine Sales Analysis

## Overview

This project focuses on analyzing historical wine sales data and forecasting future demand using time series modeling techniques.

## Objective

To identify trends, seasonality, and patterns in wine sales data and provide insights to support business decision-making and improve sales performance.

## Work Done

* Performed data preprocessing and handled missing values
* Conducted exploratory data analysis to identify trends and seasonality
* Built and compared multiple forecasting models including:

  * Moving Average
  * Exponential Smoothing
  * ARIMA and SARIMA
* Selected the best model based on performance metrics

## Key Insights

* Strong seasonality observed with peak sales during certain months
* Some product categories showed declining trends over time
* Opportunities identified for targeted marketing and promotions
